## Advanced Calculations

### Level of Detail (LOD) Expressions

LOD expressions allow you to compute values at different levels of granularity:

- **FIXED**: Calculate at a specified dimension level
- **INCLUDE**: Include additional dimensions in calculations
- **EXCLUDE**: Remove dimensions from calculations

Example use cases:
- Year-over-year comparisons
- Cohort analysis
- Customer lifetime value
- Market share calculations

### Table Calculations

Table calculations operate on the table structure:
- Running totals
- Percent of total
- Rank calculations
- Moving averages
- Percent difference

## Parameters and Dynamic Visualizations

### Creating Interactive Parameters

Parameters enable users to:
- Change values dynamically
- Switch between different metrics
- Adjust date ranges
- Toggle between different views

Best practices:
- Use descriptive names
- Set appropriate data types
- Provide clear value lists
- Add helpful tooltips

### Parameter Actions

Parameter actions create dynamic interactions:
- Click to change parameters
- Hover to update calculations
- Select to filter views
- Navigate between dashboards

## Performance Optimization

<p style="text-align: justify;"> 
Optimize Tableau performance:

1. **Data Source Optimization**: Use extracts, aggregations
2. **Calculation Efficiency**: Optimize LOD expressions
3. **Visualization Simplification**: Reduce mark counts
4. **Filter Optimization**: Use context filters strategically
5. **Dashboard Layout**: Minimize sheet count
6. **Server Configuration**: Optimize server settings

Performance is crucial for user experience, especially with large datasets.
</p>
