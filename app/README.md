# App – Jekyll + Flask

Ce dossier contient :

- **Site Jekyll** (frontend) : `_config.yml`, `_layouts/`, `_includes/`, `_posts/`, `assets/`, `index.html`, etc.
- **API Flask** (backend) : `src/`, `run.py`, `requirements.txt`

## Démarrage rapide

- **Jekyll** : `bundle install` puis `bundle exec jekyll serve` → http://localhost:4000  
- **Flask** : depuis la racine du repo, voir le [README principal](../README.md) et [Tester avant prod](../docs/TESTER_AVANT_PROD.md).

Documentation complète à la **racine du dépôt** : [README.md](../README.md), [docs/](../docs/).
