# Code de Conduite des Contributeurs

## Notre engagement

Nous nous engageons à faire de la participation à ce projet une expérience exempte de harcèlement pour tout le monde, quel que soit l’âge, la taille, le handicap, l’origine ethnique, l’identité de genre, le niveau d’expérience, la nationalité, l’apparence, la race, la religion ou l’orientation sexuelle.

## Nos standards

* Utiliser un langage accueillant et inclusif
* Respecter les points de vue et expériences différents
* Accepter les retours constructifs
* Se concentrer sur ce qui est meilleur pour la communauté
* Faire preuve d’empathie envers les autres membres

Les comportements inacceptables incluent notamment : l’utilisation de langage ou d’imagery sexualisés, le trolling, les commentaires insultants ou désobligeants, le harcèlement public ou privé, et toute autre conduite inappropriée dans un contexte professionnel.

## Application

Les mainteneurs du projet ont le droit et la responsabilité de supprimer, modifier ou rejeter les commentaires, commits, code et autres contributions qui ne respectent pas ce code de conduite.

## Version complète

Ce document est une version courte. Le **Code de Conduite des Contributeurs** (Contributor Covenant) version 2.1 est disponible en français ici :

**https://www.contributor-covenant.org/fr/version/2/1/code_of_conduct/**

En participant à ce projet, vous acceptez de respecter ce code de conduite.
